Steam客户端不识别PS5手柄不识别SwitchPro手柄不识别Xbox和国产手柄

更新于 2026-03-30 分类于 Steam 阅读次数： 阅读时长 ≈ 8 分钟

解决Steam最常见的Windows客户端之特定游戏不识别PS5手柄(或者Switch Pro手柄, 或Xbox手柄), 实现Windows中的Steam可使用各种游戏手柄来控制

有什么用

解决Steam最常见的Windows客户端之特定游戏, 不识别PS5手柄, 或者Switch Pro手柄, 或Xbox手柄, 等等无线手柄

电脑的Steam客户端, 使用无线手柄玩游戏, 实现了和游戏主机类似的功能

本文具体举例了如何设置后, 让Steam中的艾尔登法环游戏可以使用Switch Pro手柄来控制

相关内容

实现方法

根据不同的游戏客户端, 设置稍有不同; 案例征集中….

老头环适配Switch Pro手柄

艾尔登法环

2、设置手柄

非Xbox360手柄可以使用xoutput这个软件模拟。

github下载免费。怎么设置。。。B站教程一堆，此处不写了。

有些古董手柄没有home键，可以用键盘随便替代一个，这个无所谓的，基本按键有就行了。

设置完后记得运行。

重要更新：
今天测试了新的蓝牙手柄。发现又不行了。最后发现关键点就是，老头环只支持1个控制设备，这是核心思想。
所以点开电脑的设置，其它设备。如果有别的设备，是手柄的话，就删除。总之只留1个就行了。

Install emulation software

Install one of the two libraries. ViGEm (preferred) or SCPToolkit (unsupported, legacy)

Software State Download
ViGEmBus Deprecated since Nov 2, 2023 ViGEmBus

Install dependencies

Install:

  • DirectX (any version that contains dinput8.dll)
  • all the drivers for your controllers
  • .NET 7 ASP.NET and Desktop runtime
  • if you have issues, install the official XBox 360 controller drivers.

Download XOutput

  • Download the latest stable release
  • Unzip to any directory 直接解压后 就可以使用

设置XOutput

识别到switch pro手柄后

设置 中开启 GUID, 在切换语言到中文简体的 往下二个选项(倒数第二个选项), 选中.

  1. 添加控制器
  2. 然后配置各个按键的映射
  3. 保存后, 点击启用, 启用模拟XBox360手柄

完成switch pro 配置后, 开启Steam

然后在steam老头环的控制器设置中 关闭Steam输入; 是的! 关闭控制器的Steam输入

关闭控制器的Steam输入

然后 老头环就可以用Switch Pro手柄来控制了(模拟成了XBox手柄)

在老头环中的游戏设置

把控制 从 鼠标键盘, 改成控制器; 设置中的 第三个选项

本章节来源: Get Playstation icons in Red Dead Redemption on PC

Solution: We will install Lenny’s Mode Loader to make use of the PlayStation Icons Replacement

  1. Go to https://www.rdr2mods.com and download Lenny’s Mod Loader
  2. Before unzipping the files, locate your Red Dead Redemption game folder. It is most commonly found here: C:\Program Files (x86)\Steam\steamapps\common\Red Dead Redemption 2
  3. Open the downloaded zip file, go to the Mod Loader folder and extract the files to your game folder. The files need to be in the root directory of Red Dead Redemption 2 or it won’t work
  4. Next, we’ll need to install Scripthook from http://www.dev-c.com/rdr2/scripthookrdr2
  5. Once that’s downloaded, unzip wherever you’d like, then copy the contents of the “bin” folder, not the folder itself, to your game folder. Similar to what you did with lennys mod loader.
  6. Now that the mod loader and scripthook are both installed, we can download the icons. Go to Nexus Mods and download the PS4 or PS5 version of the icons, whichever you need. https://www.nexusmods.com/reddeadrede...
  7. Unzip wherever you’d like, then copy the “PS Buttons Dualsense” folder into the “lml” folder that’s located withint he Red Dead Redemption 2 folder: C:\Program Files (x86)\Steam\steamapps\common\Red Dead Redemption 2\lml This would be the same location that you placed lennys mod loader and scripthook in, but then one more folder in, lml (I might have mixed that up in the video).
  8. Assuming you followed the first video of how to use your Playstation controllers on the PC for Red Dead Redemption 2, you can open the game and you should be ready to go.
  9. Links from the video: Common folder location for Red Dead Redemption 2: 📁 C:\Program Files (x86)\Steam\steamapps\common\Red Dead Redemption 2
