M1 MacBook Pro安装Kali Linux 2024.2

安装日期: 2024.9.4

https://cdimage.kali.org/kali-2024.2/kali-linux-2024.2-live-arm64.iso

  1. 下载UTM
  2. 下载上面的Kali Linux 2024.2
  3. 新建UTM虚拟机, 选第一个默认的ARM处理器; 一些注意事项
    1. 新建整个Kali linux 的虚拟机后, 不要启用处理器的加速
    2. CDROM指向上面的kali Linux 下载到的安装ISO镜像文件
    3. CPU核心选择4; 内存默认4G, 硬盘最低要30G
    4. 注意: 虚拟机新建好以后, 需要到编辑界面, 添加硬件, seria, 把字体调节到20以上.
    5. 需要添加这个‘序列’ 显示器, 才能正常开始安装Kali Linux
    6. 安装好以后, 删除这个seria序列显示器即可
