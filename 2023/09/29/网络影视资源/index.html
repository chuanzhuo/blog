背景

经常遇到那个影视网站可以在线看电影之类的需求， 要么是追求'速食主义'（不想去下载到本地然后再播放）， 要么是想要确保影片的高清晰度高质量播放。 目前，在电视上播放影视资源（电影、电视剧）成为潮流， 另外一个方式就是在手机（以安卓手机为例）上，快速播放高清长视频。

它们共同都涉及一个技术方式：IPTV，常见视频流有：
VOD点播，在线实时直播。分享最近几年成熟的APP方案：TVBOX

这个开源的应用可以运行在各个构架的平台上：安卓手机，安卓机顶盒，Windows电脑，苹果电脑等等。
它只是一个用来播放媒体的壳（这也是为什么它还能存活的原因），
它并不包含实实在在的影视资源（直播源，m3u，m3u8地址等等）。
你可以理解它仅仅是一个电动工具，没有电，它无法开心地工作。

那么既然有了播放工具下一个问题就变成：怎么找到播放源？
播放源分为：
1。 直播源（比如iptv直播的各个电视台卫视节目，或者各大网络平台的直播）
2。 点播源，一般有两种展现形式：
一种是通过列表一页页的显示给用户来选取（那一部特定的电影），
另一种是通过搜索跨越不同的点播站/源，再展现与关键字相关的搜索结果。

今天主要分享安卓和N1机顶盒上的TVBox应用及视频源的使用经验。
以下主要以安卓手机为例，展开具体思路（在电视机顶盒上原理也一样）

材料清单

安卓手机 梯子 APP下载：https://www.123pan.com/s/UKPrVv-f06zh.html 提取码:nYTC (手机版本:mobilie，电视版本：leanback)

安卓手机安装与设置

我的安卓手机下载到的对应的APP名称为：mobile-java-arm64-v8a.apk 安装后打开APP》右下角设置按钮》点播 这里就是输入TVBOX的源， 举例说明：https://ghproxy.com/https://raw.githubusercontent.com/gaotianliuyun/gao/master/js.json

如果这个源是有效的，保存后，点击回到APP的左下角》搜索框，输入想要观看的影片名。
在出现的结果列表中选择一部影视资源，即可打开播放。

TVBOX配置去那里找

这变成关键的问题，好比有了一个好燃气灶，假如燃气输入，那燃气灶变成摆设。 接口的维护和运营是要花费时间硬件等等成本的，还涉及到遭封杀或恶意攻击等等现实遭遇。 对于普通用户，如何快速找到可用的接口，以下提供两个思路（都要借助与梯子，才能灵活应用）

思路一：github tvbox关键字
比如：https://github.com/search?q=tvbox%E9%85%8D%E7%BD%AE&type=repositories&s=updated&o=desc
个人有相关能力的用户无私分享是普通用户获取配置最直接的白piao方式，
一般来说最新commit的配置，一般通过了作者最近的测试。
另外需要在Github Raw地址前加上一些中转地址，比如https://ghproxy.com/，
用来给设备绕开gihub被屏蔽的尴尬。
举例说明#1：
anaer/Meow 自用tvbox配置
这是今天最近一条的搜索结果，打开后，
依据说明中的链接得到：https://anaer.github.io/Meow/meow.json
或者：https://raw.githubusercontent.com/anaer/Meow/main/meow.json
就是最新的接口了，不同的作者和源侧重点不一样。
比如这个接口就包含很多在线实时直播的iptv源url地址，它很适合用来看iptv电视台的电视

举例说明#2：
有一种点播站的源是从各个在线影视网站中提取影视资源的地址。
讲人话：可以搜索已知收集到的很多PHP视频站的资源，比如：
https://xiaobaotv.net/index.php/vod/play/id/40759/sid/1/nid/1.html
正在播放：侏罗纪世界3-1080P
就是可以直接在tvbox中使用，这个思路可太疯狂了。

思路二：多线路（接口）
比如：https://gitlab.com/duomv/apps/-/raw/main/duo.json
影视多线配置接口，就好比把多个的思路组合在一起，
其中某个线路挂掉之后，另外的线路就自动补充上来。
可以自己找到自己喜欢的线路后配置到一起，成为一个json文件，在APP中可以切换使用。
{
“urls”: [
{
“url”: “http://yydsys.top/duo“,
“name”: “🏡应用多多家庭版”
},
{
“url”: “http://cdn.yydsys.top/duo“,
“name”: “🏡应用多多备用”
},
]
}
