打印相片, 如何从安卓手机发送大量照片到Mac电脑

背景

打印照片一向是让我感觉头疼的事，被拖了几年后，我又一次重启了这项有益增进家庭情感的事务；遇到的主要挑战有：

  1. 手机是多台旧Android安卓系统手机，电脑是Mac M1苹果电脑
  2. 照片打印服务商一般使用App，网盘，或者邮箱来传输原图照片
    1. 打算把照片压缩到屈指可数的压缩文件，然后选择我认为最轻量级的Email邮箱发送方式
  3. 需要一次在手机上预览几千张照片，选择几百张需要打印的照片，快速地将照片传输到电脑在上传到邮箱附件。
  4. 发现有两个Mac上的APP可以从USB快速地将文件从安卓手机传输到Mac电脑：Android File Transfer（已测试可用） 和 OpenMTP（待测试）
    1. 发现这两个APP都是无法实现：预览手机中照片的同时，选取指定的多张照片来传输到MAC电脑；后来查询的原因主要指向出于Performace性能方面考虑
    2. 选择用安卓手机中默认的图片浏览器来批量浏览需要打印的照片，又无法实现将选取的多张照片压缩成一个文件

工具

  • 手机中： Root Explorer 
  • 电脑中： Android File Transfer

实现过程

  1. 使用了上一期将的安卓手机APP: Root Explorer 来轻松实现：预览照片（缩略图）和批量选取特定照片，并压缩成ZIP文件包。
    1. 批量压缩的照片，比如50张为一个ZIP压缩文件，会保存在安卓手机目录中。
  2. 最后连接手机到Mac电脑拓展的USB接口，手机中选择USB连接意图：USB浏览文件（类似的选项），
  3. 打开Android File Transfer，找到ZIP压缩文件所保存的目录，下载压缩文件到Mac电脑中。
  4. 浏览器打开您所使用的Email地址，把电脑上的ZIP压缩文件上传为附件，发送邮件

这样做除了第一步决定和选取需要打印的照片比较费时，其他步骤，速度非常流畅，迅速，还很方便。
