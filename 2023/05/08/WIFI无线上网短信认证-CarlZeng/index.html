公网IP上的服务器

  1. dns2tcp 

    1. 当LELO的路由器系统默认包中没有这个c软件，必须用github上面的版本clone后编译，或者使用下载列表中对应CPU构架已编译好的app到系统中。 【下载详见感谢列表dns2tcp-mipsel】
    2. 安装和执行步骤
      1. scp /Users/carlzeng/Downloads/dns2tcp-mipsel root@192.168.6.1:/bin
        1. 输出 dns2tcp-mipsel 100% 106KB 236.5KB/s 00:00
      2. ssh root@192.168.6.1 

        1. find -name ‘dns2tcp*‘

          1. 输出./bin/dns2tcp-mipsel

        2. chmod a+x /bin/dns2tcp-mipsel

        3. /bin/dns2tcp-mipsel

          [parse_command_args] missing option: ‘-L’

          usage: dns2tcp <-L listen> <-R remote> [-s syncnt] [-6rafvVh]

          -L <ip#port> udp listen address, this is required

          -R <ip#port> tcp remote address, this is required

          -s             set TCP_SYNCNT(max) for remote socket

          -6 enable IPV6_V6ONLY for listen socket

          -r enable SO_REUSEPORT for listen socket

          -a enable TCP_QUICKACK for remote socket

          -f enable TCP_FASTOPEN for remote socket

          -v print verbose log, default:

          -V print version number of dns2tcp and exit

          -h print help information of dns2tcp and exit

          bug report: https://github.com/zfl9/dns2tcp. email: zfl9.com@gmail.com
