How to Enable Steam Deck Battery Storage Mode

Use this guide to put a Steam Deck into battery storage mode from within its BIOS.

Before opening up a Steam Deck to perform internal repairs, Valve recommends enabling battery storage mode for increased safety and to minimize any risk of damage to the device.

If you’re enabling battery storage mode for long term device storage instead of repairs, discharge (or charge) your battery to 50-60% before enabling storage mode.

When in storage mode, your Steam Deck won’t respond to power button presses. To exit storage mode, plug your Steam Deck into the charger.

Steam Deck 恢复指示说明

  1. 点击此处下载恢复映像。

  2. 准备一个装有恢复映像的 USB 启动盘 （容量至少为 8GB）：

    • 在Windows 上，我们建议使用

      Rufus 实用工具

      。 选择恢复文件并将其写入 U 盘（此操作将格式化该 U 盘里的内容）。 完成后，选择“关闭”并弹出 U 盘。

    • 在macOS上，您可以使用

      Balena Etcher

      ，将恢复文件写入您的 U 盘。

    • 在Linux 上，您也可以使用

      Balena Etcher

      将恢复文件写入您的 U 盘；或者如果您愿意，您可以直接从命令行执行此操作：

      bzcat steamdeck-recovery-4.img.bz2 | sudo dd if=/dev/stdin of=/dev/sdX oflag=sync status=progress bs=128M

      （将 /dev/sdX 设为您插入的 USB 设备）

  3. 将启动盘通过 USB-C 适配器或集线器插入 Steam Deck。

  4. 如果您的 Steam Deck 尚未关闭，请将其关闭。 按住降低音量键并按下电源键，听到提示音后放开降低音量键，机子会启动并进入“Boot Manager”（启动管理）界面。

  5. 在“Boot Manager”界面上，选择从“EFI USB Device”（您的 USB 启动盘）启动。

  6. 启动时屏幕会变暗，稍等片刻。

  7. 启动后，您将被带到桌面环境，您可以使用触控板及扳机键导航。

恢复选项

在恢复环境中有四个不同的选项。

  • Re-image Steam Deck（对 Steam Deck 重新映像）

    - 将完全恢复出厂设置。所有用户信息、安装的游戏及应用，以及操作系统均会被清除并替换为原生 SteamOS。

  • Clear local user data（清除本地用户数据）

    - 将重新格式化 Steam Deck 上的主分区。这将删除已下载的游戏和存储在此 Deck 上的所有个人内容，包括系统配置。

  • Reinstall Steam OS（重新安装 SteamOS）

    - 将在尝试保留您的游戏和个人内容的同时，在 Steam Deck 上重新安装 SteamOS。

  • Recovery tools（恢复工具）

    - 将开启可以对 Steam Deck 的启动分区进行修改的提示。

Steam Deck LCD - Windows 11 Easy Dual Boot / Triple Boot Guide via Clover - SteamOS Windows Batocera

Steam Deck LCD - Windows 11 Easy Dual Boot / Triple Boot Guide via Clover - SteamOS Windows Batocera

This has been tested on my Steam Deck LCD running SteamOS 3.5 / 3.6 and Windows 11.

The workflow is this -

  1. Download the SteamOS Recovery Image, Windows 11 ISO and Windows 11 drivers.
  2. Burn the SteamOS Recovery Image to a USB 3.0 flash drive using rufus.
  3. Burn the Windows 11 ISO to a USB 3.0 flash drive using rufus.
  4. Copy the Windows 11 drivers to the USB drive.
  5. Manually boot to the SteamOS Recovery USB to resize the internal SSD - partition 8.
  6. Create a new NTFS file system for Windows.
  7. Manually boot to the Windows 11 USB to perform the Windows 11 install on the new NTFS file system.
  8. Install the drivers, perform Windows Update to install additional drivers.
  9. Run 2 commands to configure Windows for Clover.
  10. Manually boot to SteamOS to install Clover.
  11. All done! Clover will show a nice GUI boot menu whenever you power on the Steam Deck.

If you have a Batocera sdcard, just insert it and Clover will automatically detect it.
Other Linux distros are automatically detected by Clover too - Manjaro, Ubuntu and its derivatives, Fedora, CentOS, Ventoy, etc etc.

分配Windows需要的分区

这时, 请连接拓展坞! 因为需要同时使用USB/Typec的启动盘 和 键盘, 鼠标(非必需, 但可提高效率和体验)

上面设置完成后点击关机，插入前面制作好的恢复盘，重新U盘启动进入恢复系统，进入恢复系统后，打开KDE Partition Manager，虽然前面在系统里可能也可以直接用这个来分区，不过个人还是推荐在这个恢复系统里来进行分区操作，不容易受到其他因素影响。

打开KDE分区管理器后，我们找到机器的内置硬盘，往下拉找到home分区，右键选择Resize/Move（或者按下快捷键Ctrl+R）从Home划分出Windows的分区，上面的分区不懂的话不要去调整，不然可能会导致系统无法正常启动或者工作。

留给Windows的分区，在Free space after那里设置即可，大小由你个人决定，因为我在那边主要是玩一些学习版的小游戏，把那些游戏放在TF卡上（例如GAL），正版的还是喜欢来SteamOS这边玩，这里只分了100G，在那里输入102400，点击OK即可。

划出空闲分区以后，点击软件左上角的Apply应用设置，我们就可以关机了，格式化分区的操作推荐进PE使用DiskGenius，那样可以确保4K对齐。

安装Windows系统

这时候我们再做一个PE盘用于Windows的安装，PE用什么就看自己吧，我个人使用的是WEPE，只要能运行的了DiskGenius和Dism++就OK。

将PE写入U盘，

再将DiskGenius和Dism++(默认已经有了),

再将Window的镜像也复制进U盘的根目录

接着我们选择做好的PE盘启动，进入PE运行DiskGenius，

对刚才划分出来的空闲分区建立新分区(要在右上角试图中可见, 左侧菜单由于没有卷标等看不见该新分区)，文件系统选择NTFS，并设置4K对齐，卷标可以设置成Windows方便区分，点击工具左上角保存更改。

[友情提示: 这里的设置4K对齐 是指: 在DiskGenius新建分区格式化页面时, 指定“对齐到下列扇区数的整数倍”为 4096 扇区 (2097152字节)]

格式化完成后，我们运行Dism++，选择释放镜像，

再选择镜像所在的目录和释放的目录（刚才格式化的新分区），目标映像中选择我们需要安装的版本，之前我安装和激活的是专业工作站版，这里重装也就选这个了，勾选添加引导，将Windows镜像写入。

写入完成后关机，开机后会自动进入Windows，这里需要注意在Clover安装完成之前，Windows这边不要联网，不然联网后可能会自动更新，系统更新的话可能会掉引导。

我们还是按下Shift+F10打开命令提示符，输入oobe\bypassnro.cmd，重新启动后就可以跳过联网和登录微软账号。

安装Windows驱动

这部的可选的, 根据下载到的Windows11(或者10)镜像, 已经包含了Steamdeck的驱动

安装Clover-dualboot

Windows这边基本的准备工作完成后，我们就可以关机，准备开始安装Clover了。

我们进入BIOS中的Boot Manager，此时已经可以在BIOS中选择启动哪个系统了，这时候我们双系统可以算是成功了，不过这还没完，还有最后的一步。

这样虽然看上去可以用了，但如果其中一个系统有大规模更新的话，就可能会出现掉引导的情况，所以我们还需要安装Clover，就是文章开头提到的那个项目。

https://github.com/ryanrudolfoba/SteamDeck-Clover-dualboot

我们直接选择下载ZIP即可，当然在安装了Git的情况下，也可以使用git clone的命令下载。

下载完成后把它解压，复制到U盘上，机器那边选择启动SteamOS系统，并切换至桌面。

切换到桌面以后，我们在刚才下载的那个项目文件夹下，点击鼠标右键，选择在终端中打开（快捷键Alt+Shift+F4），或者在Konsole中使用cd命令切换到此目录。

打开终端后，我们输入chmod +x install-Clover.sh，赋予那个脚本执行权限。

再输入./install-Clover.sh，运行那个脚本以安装Clover，输入root密码即可。

接着会弹出一个窗口，让你选择在Clover的GUI引导菜单中默认自动启动的操作系统，按自己需求选择，我选择的是Windows，反正后面也可以更改，等待工具自动下载Clover ISO就行。

如果从github的下载失败, 那就使用proxy来代理一下 SteamDeck的上网, 重新再试一次.

在设置代理的页面, 输入 IP地址和端口, 然后 勾选 其他的协议也都使用这个代理

左右肩键R1,L1更换步骤及注意事项

  1. 关机, 拧下后盖的8颗螺丝

  2. 拔出电源线, 使整个主板所有元器件断电

  3. 小心卸下所有的排线, 包括触摸板震动的电源

  4. 卸下R2的卡扣, 这里按键中有R2的磁铁;

    1. 消磁或者磁铁位置不对都会导致R2板机键不灵敏, 无法准确识别等故障

  5. 卸下肩键所在副板, 用隔热胶带保护好微动附近塑料原件

  6. 风枪吹的注意事项

    1. 准备好两层卫生纸垫在下方
    2. 先把微动地步漏出桌面, 用风枪预热板子的背面
    3. 预热差不多后, 把房子放在卫生纸上方(避免隔空状态下的板子散热太快而影响化锡, 温度不够不会化, 散热太快会导致温度一直不到位)
    4. 风枪吹微动的正面, 风嘴完全覆盖整个微动, 可旋转风枪来吹; 距离由远到近
    5. 用镊子尝试移动微动, 如果温度不够, 不可强拉硬拽

  7. 吹下微动后应清理板子, 把微动孔洞内的锡用吸锡器都吸干净(用加低温锡的方法来吸锡)

    1. 把5个焊点都清理干净

  8. 把新的微动放在适当位置, 用锡膏填满孔洞, 上烙铁头来加固焊锡

  9. 如需飞线则挖出板上的铜片/铜泊, 清理干净后加助焊剂. 上0.02mm飞线, 弯尖烙铁头焊接即可

  10. 最后用洗板水清洁, 上3秒绿油加固(绝缘)

  11. 装机, 用相反的步骤将板子和排线等恢复原样, 测试按键最新情况.
