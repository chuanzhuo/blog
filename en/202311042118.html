How to Watch IPTV Live Broadcasts on iPad

  1. Use Cloud Stream IPTV Player on the iPad or ISO devices, downloaded from the Apple Store
  2. Add a local area network multicast program list, or a multicast-to-unicast HTTP program list.
    1. This is an M3U file URL address, you can first place the M3U file in a location on the NAS or OP that is accessible via HTTP
    2. Then input the local area network M3U address and name (fill in as desired) into the Cloud Stream IPTV Player, such as the address of the uploaded local m3u: http://192.168.6.116:8200/unicom-LOCAL.m3u
    3. For specific ways to obtain a local area network multicast program list, see: Capturing CCTV and Mobile TV IPTV Multicast Data - Playing Multicast IPTV Data Sources
  3. After successfully adding, you can click on the corresponding TV channel program to watch (much better than nPlayer)
